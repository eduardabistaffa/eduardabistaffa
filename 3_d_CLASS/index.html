ID e CLASS

CLASS

Existem duas caracteristicas principais do class:

  • E possivel utilizar o mesmo CLASS em varios elementos diferentes
  • E possivel utilizar várias classes no MESMO elemento

ID

Existem duas caracteristicas principais do ID que são constrastantes com as caracteristicas do class:

  • Não é possivel utilizar o mesmo ID em vários elementos diferentes
  • Não é possivel utilizar VARIOS IDs no mesmo elemento
